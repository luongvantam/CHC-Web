C++ là gì?

C++ là một ngôn ngữ lập trình mạnh mẽ và phổ biến, được phát triển từ ngôn ngữ C. Nó hỗ trợ cả lập trình thủ tục và lập trình hướng đối tượng (OOP), cho phép xây dựng chương trình linh hoạt, mở rộng và dễ bảo trì. C++ thường được dùng để phát triển phần mềm hệ thống, trình biên dịch, trò chơi điện tử, ứng dụng hiệu năng cao và nhiều lĩnh vực khác.

Để bắt đầu học C++, bạn cần cài trình biên dịch như GCC hoặc Visual Studio. Sau đó, bạn có thể viết và chạy chương trình bằng các IDE như Code::Blocks, Dev-C++ hoặc Visual Studio Code.

Các lệnh cơ bản trong C++

Dưới đây là một số lệnh và cú pháp cơ bản cần nắm khi mới bắt đầu học C++:

  • In ra màn hình:
    • std::cout << "Hello, World!" << std::endl;
  • Nhập dữ liệu từ bàn phím:
    • std::cin >> tên_biến;

Ví dụ đầy đủ:

#include <iostream>
using namespace std;

int main() {
    cout << "Nhập tên của bạn: ";
    string name;
    cin >> name;
    cout << "Xin chào, " << name << "!" << endl;
    return 0;
}

Khai báo và lưu giá trị trong biến

Trong C++, để lưu giá trị, bạn cần khai báo biến với kiểu dữ liệu tương ứng:

kiểu_dữ_liệu tên_biến = giá_trị;

Ví dụ:

int age = 15;             // số nguyên
float pi = 3.14;          // số thực
char grade = 'A';         // ký tự
string name = "Code For Casio";      // chuỗi
bool passed = true;       // giá trị đúng/sai (true/false)

Cấu trúc điều kiện

Sử dụng if, else if, else để kiểm tra điều kiện:

int a = 10, b = 5;

if (a > b) {
    cout << "a lớn hơn b";
} else if (a == b) {
    cout << "a bằng b";
} else {
    cout << "a nhỏ hơn b";
}

Vòng lặp cơ bản

C++ hỗ trợ nhiều loại vòng lặp: for, while, do while. Ví dụ với vòng lặp for:

for (int i = 1; i <= 5; i++) {
    cout << "Lặp lần thứ: " << i << endl;
}

Mảng (Array) trong C++

Mảng là danh sách các phần tử cùng kiểu:

int numbers[] = {1, 2, 3, 4, 5};

// Duyệt mảng
for (int i = 0; i < 5; i++) {
    cout << numbers[i] << " ";
}

Sử dụng hàm trong C++

Hàm là khối mã thực hiện một chức năng cụ thể. Cú pháp:

kiểu_trả_về tên_hàm(tham_số) {
    // nội dung
    return giá_trị;
}

Ví dụ hàm tính tổng:

int add(int a, int b) {
    return a + b;
}

int main() {
    int sum = add(3, 4);
    cout << "Tổng là: " << sum;
    return 0;
}

Thư viện và cú pháp thường dùng

  • #include <iostream>: dùng để nhập xuất
  • using namespace std;: giúp dùng `cout`, `cin` mà không cần `std::`
  • return 0;: kết thúc chương trình

Kết luận

C++ là ngôn ngữ rất mạnh, phù hợp cả cho người mới học và lập trình viên chuyên nghiệp. Hãy bắt đầu bằng những phần cơ bản, thực hành nhiều và kiên nhẫn. Bạn có thể tìm thêm bài tập trên các trang như HackerRank, W3Schools để luyện tập thêm.
